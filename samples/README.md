# 样例演示(100% 虚构数据)

这里的文件用于**新用户快速了解本工具的产出长什么样**,**全部是虚构演示数据**——
股票名称/代码/价格/持仓/盈亏均为编造,**不是任何真实荐股或持仓,更不构成投资建议**。

| 文件 | 演示的是 | 由什么生成 |
|---|---|---|
| `复盘-演示样例.html` | **盘后复盘**图文版(暗色主题):① 荐股维度(三池命中率)+ ② 持仓维度(按成本诊断)两模块 | `skills/render-html/scripts/review_to_html.py` |
| `样例日报-演示.html` | 日报的**叙事版**(文字研判) | `skills/render-html/scripts/md_to_html.py` ← `reports/样例日报-离线演示数据.md` |

双击 HTML 即可在浏览器打开(交互图表版需联网加载 ECharts;这两个样例是静态的,无需联网)。

想看真实数据的效果,请在**你自己的电脑**上按根目录 `README.md`「🚀 快速上手」跑起来——
你的真实持仓/交易/报告都只留在本机(`portfolio/`、`trades/`、`reports/` 已被 `.gitignore` 排除,**不会上传**)。
