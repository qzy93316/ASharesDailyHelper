> 来源:`knowledge/技术层面的--均线密码参考.pdf`(投资顾问 商建伟 A0670626040022,市场公开信息整理)。
> 蒸馏为结构化规则供代码调用,**AI 零计算**:命中条件由代码判定,AI 只解读。**仅供研究参考,不构成投资建议。**
> 原图页:`knowledge/kb/raw/ma_pages/p001–p005.png`。

## 0. 核心口诀(5日均线交易法 28 字)

> **5日均线分强弱 · 不破5日线继续做 · 跌破5日减半仓 · 3天不回全抛光**

均线角色定义(全篇纪律的锚):
- **MA5 / MA10 = 个股短期强弱"生命线"**(短线纪律看这两条)
- **MA20 / MA60 = 个股中期趋势"生命线"**(波段纪律看这两条)

## 1. 短线买点(知识点①⑧)

| 编号 | 买点条件 | 备注 |
|---|---|---|
| B1 | 股价**回踩 MA5 或 MA10**(或平台支撑)企稳 | **切记不追高** —— 好买点在回踩,不在冲高 |
| B2 | 前提:股价**站上 MA5 且站上 MA10**、且**均线多头排列**(MA5>MA10>MA20) | 只在多头结构里做买点 |
| B3 | "龙回头"型:回调 **4 / 7 / 10 个交易日**时 | 强势股回调计交易日数 |
| B4 | 低位【首板/2板】N 型涨停战法:**回踩 MA5/MA10** 买入 | 成功=当日涨幅>5% 或涨停不连板;失败=跌破 MA10 |

短期止损锚:走坏时短线止损点一般在 **MA10**。

## 2. "5 个点"纪律 —— 高抛低吸做 T(知识点②)

盘中出现以下任一,技术上一般**高抛低吸做 T**,后续看 MA5:
- T1 涨停之后、第二天**没有连板**的
- T2 当日**涨幅超过 5%** 但**没有涨停**的(看盘中涨幅,**不看收盘价**)
- T3 **偏离 MA5 超过 5%** 但没有涨停的(可反推 MA5+5% 的价位)

## 3. 短线卖点(知识点②③)

| 编号 | 卖点条件 | 口径 |
|---|---|---|
| S1 | 当日涨幅 >5% 但没涨停 | **盘中**触发,不看收盘 |
| S2 | 偏离 MA5 >5% 但没涨停 | 反推 MA5×1.05 价位 |
| S3 | 当天涨停 → 第二天基本有溢价 | 短线次日考虑是否卖出 |
| S4 | 当天涨停 → 第二天**没连板** | = 卖点,正常回踩 MA5(不连板走) |

## 4. 涨停/连板处理(知识点③⑦⑨)

- **连板留,不连板走**;后续看 MA5。
- 以下涨停按"**不连板/弱**"对待(次日多为卖点):**烂板**、**14:40 之后才涨停**、**反复打开涨停**、**封单量较小**。
- **不追首板 1 进 2**:首板晋级 2 板的成功率 100 里不足 10(现约 5–10 个),多走 N 字 → 等回踩 MA5/MA10 看能否走**反包**;弱转强走成龙头是少数。
- 高位的涨停:看到的都是**风险**。

## 5. 波段趋势票纪律(知识点④)—— 适用均线多头排列的股票

**卖点(阶梯减仓,核心):**

| 编号 | 触发 | 动作 |
|---|---|---|
| W-S1 | **跌破 MA5** | **减半仓**(短期趋势走弱) |
| W-S2 | **跌破 MA10** | **全部减仓,锁定利润**(短期生命线破) |
| W-S3 | **跌破 MA60** | **卖出**(中期走坏) |

**买点(阶梯建仓):**
- 底部**配合放量** + **连续 3 个交易日站稳 MA60** → 先打**底仓**。
- 回踩 **MA20** 二次确认 → 考虑分配**建仓**。

## 6. 仓位纪律(知识点⑤)—— 大盘均线定基准仓

| 大盘状态 | 建议仓位 |
|---|---|
| 大盘在 **MA5 且 MA10 上方**、且均线多头排列 | **5–8 成**,剩余作加减仓机动 |
| 大盘在 **MA5、MA10 下方**(无赚钱题材) | **3–5 成 / 空仓** |

## 7. 补仓与止损意识(知识点⑤补充 · 页4)

- 走势不及预期**要有止损意识**,以免**越亏越多**;等后续**重新回到 MA5** 再做关注。
- **想补仓,先看大盘**:大盘出现**止跌信号**(**中阳线** + **MA5 在 MA10 上方**、均线多头排列)才考虑补。

## 8. 箱体震荡(知识点⑥)

- 买点:回到**下箱体** / **突破上箱体**。
- 卖点:到**上箱体** / **跌破下箱体**。

## 9. 事件驱动节奏(配图:潜伏/爆发/兑现)

| 阶段 | 应对 |
|---|---|
| 潜伏期(**最安全**) | 分仓低吸、分批介入 |
| 爆发期(**可追涨**) | 去弱留强、仓位聚焦 |
| 兑现期(**要提前**) | 放量滞涨即提前出局 |

## 编程化触发条件(供 `judge.py`/`diagnose_portfolio.py`/`action_plan.py` 调用)

代码可直接判定的原子条件(close/ma5/ma10/ma20/ma60/bias5/涨幅 均已在 `indicators.py` 产出):

```
# 短线/波段 阶梯减仓(持仓端)
close < ma5                      → 短期走弱,减半仓信号(W-S1)
close < ma10                     → 短线生命线破,全减/清仓锁利信号(W-S2)
close < ma5 连续 3 个交易日        → "3天不回全抛光"(28字诀)
close < ma60                     → 中期走坏,卖出信号(W-S3)

# 高抛做 T(持仓端/盘中)
0 < 当日涨幅 < 涨停阈 且 当日涨幅 > 5%   → 高抛做T(T2/S1)
bias5 > 5% 且 未涨停                    → 高抛做T,反推 ma5*1.05(T3/S2)

# 买点(荐股/加仓端)
ma5>ma10>ma20 且 close 回踩至 ma5/ma10 附近不破  → 短线买点(B1/B2)
底部放量 且 close>ma60 连续 3 日                  → 波段底仓(W 买点)

# 仓位(组合端,需大盘指数的均线)
大盘 close>ma5 且 >ma10 且多头排列 → 基准 5–8 成
大盘 close<ma5 且 <ma10           → 3–5 成/空仓
```

> 注:`bias5 > 5%` 现已是 `hard_rules.max_bias5_pct` 的**不追高**阈值(禁买),本规则把同一阈值**反用为持仓端高抛信号**,方向互补。
> 与结构化止损(`judge.py structural_stop`,支撑位-缓冲、最小 3.5% 容错)分属两层:**均线阶梯=减仓节奏(先减半再全减),结构止损=清仓底线**。
