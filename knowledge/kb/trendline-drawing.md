# 画线操盘法(支撑/压力/趋势通道)

> 来源:`knowledge/画线操盘法.pdf`。蒸馏为结构化规则供代码调用(**AI 零计算**)。仅供研究参考,不构成投资建议。
> 大道至简:指标可信不可依赖,支撑/压力/通道的画线是操作基石。

## 五种形态

| 形态 | 确认 | 操作 |
|---|---|---|
| **强支撑** | 回落到同一位置反复止跌反弹,连接各次**回调低点**成线;需 **≥2 次**低点确认,触点越多越强 | 回落至支撑 → 抢反弹进场/加仓 |
| **强压力** | 反弹到同一位置反复受压回落,连接各次**反弹高点**成线;需 **≥2 次**高点确认 | 反弹至压力 → 减仓/了结 |
| **平行通道(箱体)** | 强压力线 + 强支撑线,筹码在中间运行 | 回落支撑买、反弹压力卖,高抛低吸波段 |
| **上升通道** | 回调低点**不断升高**,下方"上升趋势支撑线"不破则通道维持 | 回落至上升支撑线 → 买/加;**通道不破不卖** |
| **下降通道** | 反弹高点**不断降低**,上方"下降趋势压力线"不破则通道维持 | 反弹至下降压力线 → 减/清;弱势少动 |

## 主力骗线(重点)

- **诱空(破强支撑)**:跌破一直以来的强支撑时**不要急着清仓**——继续持有 **3 个交易日**。若之后**连续 2 日仍加速下跌 → 第 3 日止损**(该认错就认错,勿死扛);若跌势 **2 日内很快止住 → 拿住**,等反弹回强支撑上方再决定。
- **诱多(破强压力)**:突破强压力时**不要急着加仓**——等回调确认**压力转支撑**的有效性。强压力被突破一般须**放大量(甚至天量)**;**突破无量 = 主力拉升/骗线**嫌疑,切忌看涨追涨,谨防补仓被套。

## 编程化触发条件(供 chan/scoring/judge/watch 调用)

```
# 强支撑/压力:笔/摆动极值聚类(≥2 触点,价位相近)——chan._support_resistance 已给最近支撑/压力,
#   但未计"触点次数/强度",可扩展为多触点聚类以量化"强"。
支撑触点数 >= 2 且 现价回落贴近支撑   → 抢反弹买点(画线 B)
压力触点数 >= 2 且 现价反弹贴近压力   → 减仓点(画线 S)
# 通道方向(与 chan segments/线段方向、回调低点序列一致)
回调低点序列递增                      → 上升通道(回踩支撑线加仓;通道不破不卖)
反弹高点序列递减                      → 下降通道(反弹压力线减/清)
# 骗线过滤(与 kb/ma-code-rules 的"破位持 3 日"一致)
破强支撑后:持 3 日;续跌 2 日→第 3 日止损;2 日内止住→持有        (诱空过滤)
破强压力但当日量比 < 1(无量突破)     → 疑似骗线,不追                (诱多过滤)
```

> 注:与 [[ma-code-rules]] 的"3 天不回全抛光""跌破支撑先持 3 日"口径一致,可统一为**破位缓冲纪律**;与结构止损(`judge.structural_stop`)配合——结构止损即"强支撑-缓冲"的代码化。"无量突破不追"可作 scoring/盘中池的**突破真伪过滤**。相关:[[gap-theory]] [[volume-price-relations]] [[stop-loss-two-layer]]
