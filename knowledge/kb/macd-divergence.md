# MACD 背离使用法

> 来源:`knowledge/MACD 背离使用法.doc`。蒸馏为结构化规则供代码调用(**AI 零计算**)。仅供研究参考,不构成投资建议。
> 缠论背驰(1B/1S)已由 `chan.py` 计算并输出(`divergence`),本文件是其**通俗背离口径**的补充与选股应用说明。

## 定义

背离(背驰):股价/指数不断创新低(高),而技术指标不跟随创新低(高),即 **MACD 图形走势与 K 线走势方向相反**。

## 两类背离

| 类型 | 判定 | 特点 | 适用 |
|---|---|---|---|
| **柱状形态背离** | 股价创新低,但 MACD 绿柱一底比一底高(顶背离反之:红柱一顶比一顶低) | **敏感**,信号早 | 短线 |
| **DIF 线背离** | 股价创新低,但 DIF(白线)未低于前低(顶背离反之) | **迟钝**,但金叉/死叉确认后**把握大** | 稳健 |

- **底背离** = 价新低 + 指标未新低 → 预示低位可能**反转向上**(抄底信号)。
- **顶背离** = 价新高 + 指标未新高 → 预示高位可能**反转向下**(逃顶信号)。

## 使用要点

1. 顶底背离逃顶抄底**准确率相当高**。
2. 背离**只显示价格方向,不显示空间与时间**(可能钝化、反复背离)——需配合确认信号(金叉/死叉、量能)再动手,不可仅凭背离满仓。
3. 适合不同周期。
4. MACD 主打**稳健**(比 KDJ 钝);激进者可配 KDJ 等一起用。

## 编程化触发条件(供 scoring/judge/signals 调用)

```
# 底背离(反转候选,左侧):价创近 N 日新低,但 MACD 柱/DIF 未创新低
close == min(close[-N:]) 且 macd_bar > min(macd_bar[-N:])   → 柱状底背离(短线敏感)
close == min(close[-N:]) 且 dif      > min(dif[-N:])         → DIF底背离(钝但可靠,配金叉更佳)
# 顶背离(反转风险,减仓):镜像
close == max(close[-N:]) 且 macd_bar < max(macd_bar[-N:])   → 柱状顶背离
```

> 注:`chan.py` 的 `divergence`(1B 底背驰 / 1S 顶背驰)已是本规则的一种实现,`judge._signal_notes` 已消费为"缠论底/顶背驰"。**选股初筛(scoring)目前未把底背离作为反转候选纳入**——见 `kb/` 相关讨论:趋势追涨型评分会把底背离这类**左侧高价值反转**筛掉。相关:[[volume-price-relations]] [[gap-theory]]
