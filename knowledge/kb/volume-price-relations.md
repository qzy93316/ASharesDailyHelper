# 量价关系判断趋势(8 种)

> 来源:`knowledge/量价关系判断趋势.doc`。蒸馏为结构化规则供代码调用(**AI 零计算**)。仅供研究参考,不构成投资建议。
> 量=单位时间成交量(人气/主力动态镜子);价=以收盘价为准。

## 八种量价组合

| # | 组合 | 位置/背景 | 含义 | 信号 |
|---|---|---|---|---|
| 1 | **量增价平** | 低位(经长跌) | 放量而价不涨 = 新资金打压建仓;一旦放量掉头向上则底部形成 | 🟢 转阳信号(底部候选) |
| 2 | **量增价升** | 底部盘整后 | 利好聚集、换手活跃,量价同升 | 🟢 买入信号 |
| 3 | **量减价升** | 持续上升中 | 缩量涨=主力控盘高、筹码锁定;但属量价背离,若再放量恐高位出货 | 🟡 持有(警惕再放量) |
| 4 | **量平价升** | 多头初升段 | 量平价涨,留意主力资金是否不足 | 🔵 持续买入 |
| 5 | **量减价平** | — | 主力乏力、认同度下降、接盘减弱 | ⚠️ 警戒,观望 |
| 6 | **量减价跌** | 下跌/上升中期 | 缩量跌,跌势将维持 | ⚪ 持币观望 |
| 7 | **量平价跌** | — | 未形成一致看空,多为控盘主力**逐渐退出前兆** | 🔻 继续卖出 |
| 8 | **量增价跌** | 高位(经大涨) | 获利筹码抛售,高位放量下跌 | 🔴 卖出/出局 |

## 关键提炼

- **低位量增价平 / 量增价升** = 最有价值的**底部转折**信号(主力建仓/启动)。
- **高位量增价跌 / 量减价升再放量** = 出货警示。
- 缩量:上涨中缩量=控盘健康;下跌中缩量=跌势未尽。

## 编程化触发条件(供 scoring/judge 调用;量用 vol_ratio、价用 pct_chg / 位置用 bias/均线)

```
量↑ = vol_ratio >= 1.5    量平 = 0.8~1.5    量↓ = < 0.8
价↑ = pct_chg > 1%        价平 = -1%~1%     价↓ = < -1%
低位 = 空头/弱空 或 close 近 N 日低位区;高位 = 多头且 bias5 偏高/近压力

低位 + 量↑ + 价平/价升   → 底部转阳候选(反转,高价值,当前 scoring 易漏)
上升 + 量↓ + 价↑         → 控盘健康,持有
上升 + 量↑再现 + 价滞涨   → 高位出货预警
高位 + 量↑ + 价↓         → 出货,卖出
```

> 注:现有 `indicators.vol_pattern`(缩量回调/放量上涨/量能正常/放量下跌)只覆盖部分组合,且**未结合"位置(高/低位)"**——低位量增价平这类底部信号在选股初筛里没有加分通道。相关:[[macd-divergence]] [[gap-theory]] [[ma-code-rules]]
