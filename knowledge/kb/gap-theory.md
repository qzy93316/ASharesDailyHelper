# 缺口理论

> 来源:`knowledge/缺口理论 (1).docx`。蒸馏为结构化规则供代码调用(**AI 零计算**)。仅供研究参考,不构成投资建议。
> 缺口(跳空):股价快速变动中一段无成交的真空区。回到该价位称"回补/补空"。

## 四类缺口

| 类型 | 位置 | 是否回补 | 操作含义 |
|---|---|---|---|
| **普通缺口** | 密集整理区(三角/矩形内) | **很快回补** | 高抛低吸:向上普通缺口→缺口上方相对高点抛,补后再买回;向下反之 |
| **突破缺口** | 反转/整理形态完成、突破盘局 | **不易回补**(尤其放量) | 真突破标志;向上突破须**放量**;缺口越大后市越强;**确认后可大胆买入** |
| **持续/测量缺口** | 上升/下跌途中 | 通常不回补 | 中继信号;后续幅度≈从启动到本缺口的幅度(可测目标);常是一波行情第二个缺口 |
| **消耗/竭尽缺口** | 急升/急跌末段 | **几日内回补则转势** | 常伴当天最高成交量;购买力/沽压耗尽,股价很快回落/回升 |

## 关键判别

- **突破缺口 vs 假突破**:向上突破缺口若被**完全回补、价回到缺口下方 → 突破不成立**。突破前放量、缺口后缩量→短期可能回补;缺口后持续放量→短期难补。
- **向上跳空**在后续调整中常**起支撑**;**向下跳空**在后续反弹中常成**阻力**。
- **缺口个数**:一波行情缺口越多越接近终结。第 3 个缺口暗示升势将终结,第 4 个→短期下跌可能大增。第 3/4 个末期缺口即"衰竭缺口",几日内被补一般要转势。

## 早盘缺口应用口诀

- 高开,回档**不补缺**,回升**高过开盘价** → **看多**
- 高开,回档**补缺**,回升**不高于开盘价** → 看空(诱多)
- 低开,回升**补缺**,回调**不低于开盘价** → 看多(诱空)
- 低开低走 / 回升**不补缺** → **看空**

## 编程化触发条件(供 signals/盘中池/watch 调用)

```
# 缺口检测(相邻日 K)
今日最低 > 昨日最高          → 向上跳空缺口 gap_up,幅度=(今低-昨高)/昨高
今日最高 < 昨日最低          → 向下跳空缺口 gap_down
# 分类(需上下文)
放量 + 突破前期平台/形态     → 突破缺口(不补→强,可介入)
缺口后价回到缺口下沿并补齐   → 突破失败(假突破,离场)
急涨末段 + 当日天量 + 缺口   → 竭尽缺口(几日内补→转势预警)
# 早盘(盘中池/watch,用现价 vs 昨收/开盘/昨高)
高开 且 现价 > 开盘价 且 未补昨日缺口 → 看多
低开低走 或 补缺后不过开盘价          → 看空
```

> 注:缺口目前**未被检测**——`indicators`/`chan` 都没有缺口识别。突破缺口是"早期强势买点",正是趋势追涨评分**在均线尚未多头排列时会漏掉**的一类。相关:[[volume-price-relations]] [[trendline-drawing]]
